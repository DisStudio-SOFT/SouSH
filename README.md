# Компиляция шелла

```
make build-sh
```

# Компиляция команд

```
make build-commands
```

# Установка шелла

```
make build-sh
make build-commands
make install
```

Если вы хотите сделать SouSH основным шеллом:
```
chsh -s /bin/soush
```
Потом выходите и заходите в систему.(обязательно, без этого шелл не поменяется)
